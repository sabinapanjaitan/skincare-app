# 🌿 GlowCart: Skincare E-commerce App

GlowCart adalah aplikasi e-commerce modern yang difokuskan pada pengalaman berbelanja produk perawatan kulit (skincare) yang intuitif, cepat, dan estetik. Proyek ini dibangun sebagai wujud eksplorasi dalam pengembangan aplikasi *multi-platform* menggunakan Flutter.

## ✨ Key Features
- **Curated Catalog:** Menampilkan produk skincare dengan antarmuka yang bersih dan mudah dinavigasi.
- **Responsive Design:** Pengalaman pengguna yang mulus baik di perangkat mobile maupun layar web.
- **Dynamic Cart:** Sistem keranjang belanja yang interaktif untuk memudahkan pemilihan produk.
- **Modern UI:** Fokus pada estetika visual untuk meningkatkan kenyamanan pengguna.

## 🛠 Tech Stack
- **Framework:** Flutter
- **Language:** Dart
- **Tooling:** VS Code, Git/GitHub

## 🚀 Why this project?
Proyek ini dikembangkan untuk mengasah kemampuan saya dalam:
- Mengimplementasikan *state management* pada aplikasi e-commerce.
- Membangun antarmuka yang responsif dan estetik.
- Mengelola alur data pengguna dari katalog hingga ke keranjang.
